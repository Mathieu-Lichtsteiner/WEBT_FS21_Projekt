Informationen

Willkommen auf meiner Webseite!
Diese habe ich als Abschlussprojekt im Modul Webtechnologien gestaltet und eingereicht. Es geht darum eigene Bilder hochzuladen und diese mit einem lustigen MakeUp zu versehen.
Viel Spass!

Bild hochladen

Hier können Sie ein Bild auswählen, dass Sie bemalen möchten. Achtung, dabei wird Ihr bisheriges "MakeUp" zurückgesetzt!
❌

Bild zurücksetzen

Hier können Sie das Hintergrundbild löschen und auf einer weissen Leinwand malen. Falls Sie Ihr Bild wieder möchten, benutzen Sie den nächsten Knopf "MakeUp zurücksetzen".

MakeUp zurücksetzen

Hierbei wird Ihr makeUp gelöscht und auf das hochgeladene Bild zurückgesetzt.
🖉

Freihand zeichnen

Einfaches zeichnen, wie Sie es mit Farbstiften tun würden, aber halt mit der maus. Funktioniert ähnlich wie "Drag & Drop" also drücken & halten um zu zeichnen!
╱

Linie / Pfad zeichnen

Mit diesem Tool können Sie Linien zeichnen. Dabei wird ein Linienzug erstellt, bis Sie einen Doppelklick machen. Dann beginnt ein neuer Pfad.
◯

Kreis zeichnen

Mit drei Klicks können Sie bestimmen, wo ein Kreis platziert werden soll. Achtung, braucht etwas übung, um es perfekt zu machen.
⁔

Bogen zeichnen

Leider ist das bisher noch nicht Implementiert! (Zeitaufwand) Mit drei Klicks können Sie bestimmen, wo ein Bogen platziert werden soll. Dabei können nur Kreisförmige und keine elliptischen Segmente platziert werden.

▭

Rechteck zeichnen

Mittels zwei Klicks können sie die Eckpunkte eines Rechtecks bestimmen, bzw. zeichnen.
△

Dreieck zeichnen

Mit drei Klicks können Sie ein Dreieck zeichnen.

Farbe auswählen

Mit diesem unbeschrifteten Rechteck können Sie die zeichen-Farbe definieren.

MakeUp designen
